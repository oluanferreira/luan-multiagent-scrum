# License Clarification (Core vs Pro)

**Last updated:** 2026-02-15

This document clarifies the licensing boundary in Synkra LMAS.

## Scope

- `lmas-core` (this repository): **MIT**
- `lmas-pro` (private/proprietary module): separate license terms defined in that module/repository

## What This Means

- The open-source core framework is MIT-licensed and can be used under MIT terms.
- Commercial or proprietary restrictions, if any, belong to `lmas-pro` and do not change the MIT license of `lmas-core`.

## Practical Guidance

- If you are using only `lmas-core`, follow MIT terms in `LICENSE`.
- If you are using any `lmas-pro` capability, also review and comply with `lmas-pro` license terms.

## Related Files

- `LICENSE` (root): MIT license for `lmas-core`
- `docs/legal/terms.md`: terms of use for this repository
