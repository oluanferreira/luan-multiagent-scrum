## Vision General

**@pm** (Trinity) es el agente de Product Management del LMAS, responsable por:

- Estrategia de producto
- Roadmap y planificacion
- PRDs y especificaciones
- Priorizacion de features

## Comandos Principales

| Comando | Descripcion |
|---------|-------------|
| `*prd` | Crear PRD |
| `*roadmap` | Gestionar roadmap |
| `*prioritize` | Priorizar backlog |
| `*spec` | Crear especificacion |
